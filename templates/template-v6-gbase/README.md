# template-v6-gbase

iPLAT4J V6 + GBase8s 后端多模块模板（`bom/common/service/web`）。

## 固定组合

- iPLAT4J：`6.2.4.3`
- 数据库：`GBase8s`
- 驱动：`com.gbase:gbasedbtjdbc:3.3.0_2P20220224_7a4aa7`

## 数据库脚本

- `sql/gbase/iplat4j_create.sql`
- `sql/gbase/iplat4j_insert.sql`

## 快速验证

```bash
cd template-v6-gbase
mvn clean validate
```
